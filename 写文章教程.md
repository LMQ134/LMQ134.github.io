## 每次发布文章的完整流程

**第一步，打开 PowerShell，进入博客文件夹：**
```
cd my-blog
```

**第二步，新建文章文件夹和文件：**
```
mkdir content\posts\文章名
notepad content\posts\文章名\index.md
```

**第三步，把下面内容粘贴进记事本，修改标题和正文：**
```
+++
title = "文章标题"
date = 2024-01-03T00:00:00+00:00//时间的格式不能变
draft = false
description = "文章简介"
image = "cover.jpg"//可以不放图片
+++

在这里写正文内容，支持中文。

```
保存关闭记事本。

**第四步，把封面图片放进文章文件夹，命名为 `cover.jpg`**

路径是：`D:\my-blog\content\posts\文章名\`

**第五步，推送发布：**
```
git add .
git commit -m "新文章"
git push
```

等 1~2 分钟后访问 👉 https://LMQ134.github.io 就能看到新文章了 🎉

---

**记住两个注意事项：**
- 文章名和文件夹名只用**英文和数字**，不要用中文
- 文章内容可以随便写中文 😊



---

## 删除文章的方法

**第一步，删除文章文件夹：**
```
Remove-Item -Recurse -Force content\posts\文章名
```

比如删除 `my-second-post`：
```
Remove-Item -Recurse -Force content\posts\my-second-post
```

**第二步，推送到 GitHub：**
```
git add .
git commit -m "delete post"
git push
```

等 1~2 分钟后博客自动更新，文章就消失了 ✅

---

**如果只是想暂时隐藏文章，不想彻底删除：**

用记事本打开文章的 `index.md`，把 `draft = false` 改成 `draft = true`，然后推送，文章就会从博客上隐藏，但文件还在本地 😊
